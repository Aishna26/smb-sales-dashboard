# SMB Sales & Operations Dashboard
This project simulates how a small business can turn messy sales data into clear insights and forecasts using Python, SQL, and Excel. The final result is a simple dashboard designed for non-technical users to quickly understand sales performance and customer behavior.

## What This Project Shows
- Cleaning and preparing real-world business data
- Analyzing sales and customer trends
- Building a simple forecasting and segmentation model
- Delivering insights through an Excel dashboard
